# Smart Calculator (Python)

Professional command-line calculator with input validation.

## Features
- Menu-driven operations
- Safe numeric input
- Division by zero handling
- Continuous execution

## Concepts Used
- Functions
- Loops
- Exception handling
- Modular programming

## Run
python calculator.py
